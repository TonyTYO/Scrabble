Welcome to Scrabble Language Utilities

The Files Tab allows you to create a suitable subdirectory with all necessary files for the new language.
Instructions.html, Final_score.html: Translate all the text
in these two html files.
alphabet.py: Details of all the settings are in the comments
at the top. Translate or change everything that needs changing.
Change the text Language on the first line to the name
of your language - this appears in the initial language choice menu.
Finally change develop to active on the first line to make the language operational.
scrabble_letters.png: Check all the letter tiles and alter or
add as required. The positions of any extra graphics need to be added to TILE_POSITIONS in aphabet.py


The GADDAG Tab allows you to create a GADDAG data structure from a lexicon.
At least one of these is required if you wish the computer to be a player although you can have several at different levels.
The lexicon needs to be a text file with one word per line. Any accents will be stripped from the word. Any words that cannot be
processed will be printed out together with the line no.
You will need to specify an already set up language or list any
multi-character letters. If there are no multi-character letters this section may be left blank.
It will also create a text file of two letter words if required. This is not essential but can be helpful to the computer player.
