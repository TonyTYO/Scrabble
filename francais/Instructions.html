Bienvenue au Scrabble

Ceci est un jeu de scrabble normal pour deux joueurs utilisant le dictionnaire ODS5. Toutes les règles habituelles s'appliquent.

Le joueur actuel déplace les carreaux du rack sur le plateau avec la souris ou le doigt et les laisse tomber sur un carré.

Ils resteront mobiles jusqu'au le bouton Finir le tour est pressé pour terminer le tour. Le rack de l'autre joueur sera atténué.


Les boutons suivants sont disponibles pendant le jeu.

Commencer le tour: Appuyez pour commencer votre tour.
Vous ne pourrez rien faire avant d'appuyer sur.		 Finir le tour: Appuyez sur pour marquer la fin de votre tour.
Vous ne serez pas en mesure de déplacer des carreaux après avoir appuyé sur cette.
Passer le tour: Appuyez pour passer le tour. Échange: Appuyez pour échanger des carreaux.
Contestation: Activé pour l'autre joueur après l'appui Finir le tour. Appuyez dans les 5 secondes pour contester les mots. Partir: Appuyez sur ceci pour aller à la carte de score ou terminer le jeu.

Les boutons suivants sont disponibles après l'appui Partir.

Retour: Appuyez sur pour retourner au jeu. Nouvelle: Appuyez pour commencer un nouveau jeu.
Quitter: Appuyez pour quitter.
